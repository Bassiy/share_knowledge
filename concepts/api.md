# API

## 概要
プログラム同士を繋ぐ窓口の総称。ネットワーク越しだけでなく、同じPC内の機能呼び出しも含む広い概念。

## 理解したこと

- **API（窓口）** — 遠くのサーバーや同じPC内にある「関数」を呼び出すための受付
- **APIを叩く** — その受付に引数を渡して処理を依頼するアクション
- **中身はブラックボックス** — 使う側は内部がJavaだろうC#だろうが関係なく、ルール通りに叩けば結果が得られる
- **ネットワーク越しだけではない** — Win32 API（WindowsのOS機能）やUnityのAPIなど、同じPC内で完結するものも「API」と呼ぶ

### REST APIとの関係
REST APIはAPIという大きなカテゴリの中の「一種（人気なスタイル）」。  
APIが親・広い概念で、REST APIは子・特定のスタイル。
