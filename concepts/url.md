# URL（Uniform Resource Locator）

## 概要
Web上のリソースの場所を指定するための文字列。スキーム・ホスト名・パスの3要素を基本に構成される。

## 理解したこと

### 基本構成
`スキーム://[ユーザ名:パスワード@]ホスト名[:ポート番号]/パス`

| 要素 | 役割 |
|------|------|
| スキーム | 使用するプロトコル（例：https） |
| ユーザ名・パスワード | ログイン情報（省略可） |
| ホスト名 | 接続先コンピュータ（ドメイン名またはIPアドレス） |
| ポート番号 | 接続先ポート（省略可） |
| パス | サーバー内のファイルの格納位置 |

### ポイント
- ホスト名の指定：開発段階はIPアドレス、ユーザに届く段階はドメイン名が一般的
- パスを省略するとデフォルトドキュメントが返される
