# アプリケーション層の主要プロトコル

## 概要
TCP/IPのアプリケーション層で使われる代表的なプロトコル群。用途ごとに役割が分かれている。

## 理解したこと

| プロトコル | 役割 |
|-----------|------|
| HTTP/HTTPS | Web・モバイルアプリの通信 |
| SMTP | メール送信・サーバ間転送 |
| POP3 | メールボックスからメールを取り出す |
| IMAP4 | メールボックスからメールを読む |
| FTP | ファイル転送 |
| SSH | コマンドベースのリモート操作 |
| DNS | ドメイン名 ↔ IPアドレスの変換 |
| NTP | 時刻同期 |

- メールが相手オフライン時でも届くのは、送信（SMTP）と取り出し（POP3/IMAP4）が分離しているから。サーバ間の転送で完結するため、受信者の状態に依存しない非同期設計になっている
