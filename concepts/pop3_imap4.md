# POP3 / IMAP4

## 概要
メールサーバからメールを受信するためのプロトコル。SMTPが「送信」を担うのに対し、こちらは「取り出し」を担う。

## 理解したこと

### POP3 と IMAP4 の違い

| | POP3 | IMAP4 |
|---|---|---|
| メールの保存先 | ダウンロードして端末に保存 | サーバに保持したまま |
| サーバからの削除 | 取得後に削除される | 残る |
| 複数デバイス対応 | 不可（1台のみ） | 可（どの端末からでも閲覧） |
| 現在の立ち位置 | ほぼ廃れた（ニッチな需要のみ） | 主流 |

### ポート番号

| プロトコル | ポート | 備考 |
|---|---|---|
| POP3 | 110 | 暗号化なし |
| POP3S | 995 | SSL/TLS暗号化 |
| IMAP4 | 143 | 暗号化なし |
| IMAPS | 993 | SSL/TLS暗号化。現代のメインストリーム |

- 暗号化バージョン（POP3S / IMAPS）はHTTPSと同様にSSL/TLSを使用
- 現代では **IMAPS（993番）** が事実上の標準

### なぜ IMAP4 が主流になったか
スマホ・PC・タブレットなど複数デバイスを使うのが当たり前になったため。POP3はサーバからメールを削除するため、1台以外では閲覧できなくなる。
